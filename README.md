# ExitProtocol
*A virtual escape room group project created for CSCE 247: Software Engineering*

__Team__ : Emmanuel Gonzales Paz, Henry Monteith, Brayden Edwards, Jada Young, Danielle Williams


## Overview
ExitProtocol is a team-based virtual escape room application designed and developed as part of a Software Engineering course. The project focuses on applying object-oriented design, Agile/Scrum workflows, and collaborative development practices to build an interactive puzzle-solving experience.  

Players navigate through a series of challenges, interacting with game elements and solving puzzles to progress through the escape room.

## Tech Stack

* Backend: Java
* Frontend: JavaFX, Scene Builder
* Design and Prototypign: Figma, Mockitt
* Version Control and Collaboration: Git/Github
* Methodology: Scrum

## Features
* Interactive escape room gameplay with multiple puzzles
* Modular game logic supporting extensibility
* GUI built with JavaFX for user interaction
* Team-based development with iterative prototyping and feedback cycles
